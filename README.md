# Broos Home Assistant Add-ons

Repository for Home Assistant add-ons. Add this repo URL in HA to install the contained add-ons via the UI.

## Add this repository in Home Assistant
1) Copy your repo URL (e.g., `https://github.com/whizzrd/broos-ha-addons`).
2) In HA: Settings → Add-ons → Add-on Store → ⋮ → Repositories → paste URL → Add → Reload.
3) The add-ons in this repo will appear in the store list.

## Current add-ons
- **Meetpalen golfhoogte** – Rijkswaterstaat golfhoogte to MQTT/HA discovery. See `meetpalen-golfhoogte/README.md` for configuration, QA filtering, and troubleshooting.
